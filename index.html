First Commands

  • Leia o manual do git para os comandos mais usados
  • Leia o feedback do git

git help <comand>

							
	$ git help
	$ git help init

git init

							
	$ git init

git clone <repository>

							
	$ git clone https://github.com/twbs/bootstrap.git

git status

							
	$ git status

git add <filter>

							
	$ git add .

git commit <options>

							
	$ git commit -m "First commit"
